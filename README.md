# LuxeHome – Premium Home Appliances & Furniture

A stunning, fully animated e-commerce website for premium home appliances and furniture.

## Features

- 🎨 **Dark Luxury Design** — Deep navy/purple gradients with glassmorphism effects
- ✨ **Custom Cursor** — Animated magnetic cursor with hover effects
- 🎞️ **Hero Slider** — Auto-playing 3-slide hero with parallax orbs and floating cards
- 🛍️ **Shopping Cart** — Slide-in drawer with quantity controls & localStorage persistence
- ❤️ **Wishlist** — Save favourite items with slide-in drawer
- 🔍 **Smart Search** — Live overlay search with instant results
- 🏷️ **Product Filters** — Filter by category with smooth transitions
- 📦 **Product Modal** — Detailed quick-view modal with features list
- 📊 **Animated Stats** — Count-up number animations on scroll
- 💬 **Testimonials** — Auto-scrolling testimonial carousel
- 📧 **Newsletter** — Subscription form with toast notification
- 📱 **Fully Responsive** — Mobile-first with hamburger menu
- 🎬 **AOS Animations** — Scroll-triggered fade/slide/zoom animations
- 💾 **LocalStorage** — Cart and wishlist persist across sessions

## Tech Stack

- **Vite** — Fast dev server & bundler
- **Vanilla JS (ESM)** — Zero framework dependencies
- **Vanilla CSS** — Custom design system with CSS variables
- **Google Fonts** — Playfair Display + Inter
- **Font Awesome 6** — Icons

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push this repo to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Set **Build Command**: `npm run build`
4. Set **Output Directory**: `dist`
5. Deploy! 🚀
