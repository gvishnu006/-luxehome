# LuxeHome — Premium Home Appliances & Furniture Website

A stunning, fully animated e-commerce website for home appliances and luxury furniture. Built with cutting-edge web technologies featuring premium UI/UX design, glassmorphism effects, and smooth micro-animations.

![LuxeHome Preview](https://images.unsplash.com/photo-1586023492125-27b2c045efd7?w=1200&h=600&fit=crop&q=80)

## ✨ Features

### Design & UI/UX
- 🎨 **Dark & Light Theme** — Elegant theme toggle with smooth transitions
- 🖱️ **Custom Animated Cursor** — Interactive cursor with hover effects
- ✨ **Glassmorphism Design** — Modern frosted-glass UI elements
- 🎭 **Scroll Reveal Animations** — Elements animate in as you scroll
- 🧲 **Magnetic Buttons** — Buttons follow cursor movement
- 🌊 **Parallax Effects** — Immersive depth scrolling
- 🎪 **3D Card Tilt** — Interactive hero image with 3D perspective
- 💫 **Floating Particles** — Ambient animated particle system
- 📐 **Responsive Design** — Pixel-perfect on all devices

### Functionality
- 🛒 **Shopping Cart** — Full cart system with add/remove/quantity management
- 🔍 **Live Search** — Real-time product search with instant results
- 🏷️ **Product Filters** — Category-based filtering with smooth transitions
- 📊 **Animated Counters** — Statistics counter with easing animation
- 💬 **Testimonials Slider** — Auto-playing review carousel
- 📧 **Newsletter & Contact Forms** — Interactive forms with floating labels
- 📜 **Infinite Marquee** — Continuously scrolling feature banner
- 🔝 **Back to Top** — Smart scroll-to-top button

## 🛠️ Tech Stack

- **HTML5** — Semantic structure with SEO best practices
- **CSS3** — Vanilla CSS with custom properties, animations, and transitions
- **JavaScript** — Vanilla JS with no dependencies
- **Google Fonts** — Outfit + Playfair Display typography
- **Unsplash** — High-quality product imagery

## 🚀 Quick Start

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/luxehome.git
```

2. Open `index.html` in your browser, or use a local server:
```bash
npx serve .
```

3. That's it! No build step required.
